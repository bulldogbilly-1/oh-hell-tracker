# Oh Hell Tracker - Setup Instructions

## Prerequisites

Node.js 18+ is required. Download from: https://nodejs.org/

## Install & Run

Open a terminal in this folder and run:

```
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Notes

- The SQLite database (`oh-hell.db`) is created automatically on first run in the project root
- The app is mobile-first, best viewed at ~430px width
- To test on mobile, use your browser's DevTools responsive mode
